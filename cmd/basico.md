## Comandos

- dir - mostrar carpetas del directorio actual
- mkdir +"carpeta - crear carpeta
- ls- muestra todos los items de la carpeta
- cat - mostrar contenido de un archivo
- del- eliminar
- cls - limpiar la consola; no elimina el progreso
- echo - consola repite
- "aplicacion" - abrir aplicación

## Combinacion comandos

- "aplicación" + "archivo" - arir x archivo en y aplicación
- echo + "texto + > "ejemplo.txt" -crea archivo con ese contenido
- cd +"carpeta"- entrar a carpeta de
  -cd + .. - ir a carpeta anterior

## Teclas

- TAB - autocompletado

## Combinacion teclas

- CTRL+shift+T- nueva terminal
- CTRL+shift+W- cerrar terminal

## ejecutar

wt- abrir consola predeterminada

## GIT

git init - inicializar git en carpeta
git config --global user.name - definir usuario
git config --global user.email - mail
git add . - añadir archivos a cola
git commit -m "mensaje"- añadir mensaje al commit
git remote add origin + "link repositorio" - enlazar carpeta con repositorio
git push -u origin main - subir archivos
git pull
